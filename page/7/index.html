Uva11059 Maximum Product

Uva11059 Maximum Product Given a sequence of integers S = {S1, S2, . . . , Sn}, you should determine what is the value of the maximum positive product involving consecutive terms of S. If you cannot find a positive sequence, you should consider 0 as the value of the maximum product. Input Each test case starts with 1 ≤ N ≤ 18, the number of elements in a sequence. Each element Si is an integer such that ?10 ≤ Si ≤ 10. Next line will have N integers, representing the value of each element in the sequence. There is a blank line after each test case. The input is terminated by end of file (EOF). Output For each test case you must print the message: ‘Case #M: The maximum product is P.’, where M is the number of the test case, starting from 1, and P is the value of the maximum product. After each test case you must print a blank line. Sample Input 3 2 4 -3 5 2 5 -1 2 -1 Sample Output Case #1: The maximum product is 8. Case #2: The maximum product is 20. 题意：寻找乘积最大的连续子序列 由于只有18个整数，直接完全枚举即可。 注意输出的时候要%lld,因为绝对值不超过10且不超过18个数，则最大乘积不超过10^18，在long long 10^19的范围内，所以可用long long 储存答案并用lld输出 ac代码如下

#include
using namespace std;
int main(){
int n;
int s[20];
int kase=0;
while(~scanf(“%d”,&n)){
for(int i=0;i<n;i++){
scanf(“%d”,&s[i]);
}
long long max=0;
for(int i=0;i<n;i++){
long long ans=1;
for(int j=i;j<n;j++){
ans*=s[j];
if(ans>max){
max=ans;
}
}
}
printf(“Case #%d: The maximum product is %lld.\n\n”,++kase,max);
}
return 0;
}

Uva725 Division

uva725 division Write a program that finds and displays all pairs of 5-digit numbers that between them use the digits 0 through 9 once each, such that the first number divided by the second is equal to an integer N, where 2 ≤ N ≤ 79. That is, abcde fghij = N where each letter represents a different digit. The first digit of one of the numerals is allowed to be zero. Input Each line of the input file consists of a valid integer N. An input of zero is to terminate the program. Output Your program have to display ALL qualifying pairs of numerals, sorted by increasing numerator (and, of course, denominator). Your output should be in the following general form: xxxxx / xxxxx = N xxxxx / xxxxx = N . . In case there are no pairs of numerals satisfying the condition, you must write ‘There are no solutions for N.’. Separate the output for two different values of N by a blank line. Sample Input 61 62 0 Sample Output There are no solutions for 61. 79546 / 01283 = 62 94736 / 01528 = 62 思路：枚举即可，注意到abcde=n*fghij,可以只枚举abcde得到答案，当abcde和fghij合起来>10位就可以break了，因为题目要求必须刚好是0~9的排列。 可用sprintf转换，和%05d补零 注意题目要求每个n输入后的结果，有一个空行，且0终止程序，也就是说最后一次输入0的时候，什么也不要输出。所以可以把每一次输出结束后的空行,放到下一个输入处理的开头来输出，就可以避免输入0时，上一次输出结束时多输出了空行而造成PE. ac代码如下：

#include

#include

#include
using namespace std;
int main(){
int n;
int mcase=0;
while(~scanf(“%d”,&n)&&n){
int f;
char buf[99];
int cnt=0;
if(mcase++){
printf(“\n”);
}
for(int i=1234;i<=99999;i++){
f=i*n;
sprintf(buf,”%05d%05d”,f,i);
if(strlen(buf)>10){
break;
}
sort(buf,buf+10);
bool ok=1;
for(int j=0;j<10;j++){
if(buf[j]!=’0’+j){
ok=0;
break;
}
}
if(ok){
cnt++;
printf(“%05d / %05d = %d\n”,f,i,n);
}
}
if(!cnt){
printf(“There are no solutions for %d.\n”,n);
}
}
return 0;
}
