Best Time to Buy and Sell Stock II

Posted on 2018-08-09 | In 算法 , 贪心

Description

Say you have an array for which the ith element is the price of a given stock on day i.

Design an algorithm to find the maximum profit. You may complete as many transactions as you like (i.e., buy one and sell one share of the stock multiple times).

Note: You may not engage in multiple transactions at the same time (i.e., you must sell the stock before you buy again).

Solution

My basic thought is that I use two cursors to record buy index and sell index. We traverse the entire prices array, every time when the j + 1 one bigger than the j one, which means the sequential is increased, we increase j cursor. When the j + 1 one is less or equal than the j one, we calculate the profit we make in this period and add it to the toal profit, then assign i to j + 1 so that we can move on to next peroid. One edge case is that the sequence has no decline period. To sovle this case we should do an extra judge that if until the last one the sequence is keep increasing, we calculate the last period’s profit and add into total profit. (This logical maybe can be simplified by we add a 0 in the tail of prices array in advance so that we can make the whole judge process uniform.)

So, we can find that everytime when the j + 1 is one bigger than the j one, we simply get the differential between them and add it to the total profit, after traversing the entire array we can get a decent total profit.

The code is given below:

class Solution:
    def maxProfit(self, prices):
        """
        :type prices: List[int]
        :rtype: int
        """
        ans = 0
        for i in range(1, len(prices)):
            if prices[i - 1] < prices[i]:
                ans += (prices[i] - prices[i-1])
        return ans
